# .github | Global Defaults

This repository is a central hub for my GitHub account's **Community Health Files**. It provides a consistent set of guidelines and templates for all my public projects.

---

## Repository Structure

The following files are managed here:

*   **`CONTRIBUTING.md`**: Guidelines for developers who want to contribute to my projects.
*   **`SECURITY.md`**: Instructions on how to responsibly report security vulnerabilities.
*   **`pull_request_template.md`**: A standardized checklist that appears whenever someone opens a Pull Request.
*   **`.github/ISSUE_TEMPLATE/`**: 
    *   `bug.yml`: A structured form for reporting bugs.
    *   `feature.yml`: A structured form for suggesting new features or improvements.
